## 6. Автотесты для Demoblaze (Python + Playwright)

Написала автотесты для интернет-магазина Demoblaze с нуля.  
Код на Python с использованием библиотеки Playwright.  
Тесты автоматизируют проверки, которые раньше выполнялись вручную.

Папка: Demoblaze.com, и папка внутри репозитория 'autotests'.

**Структура проекта:**

- один файл `autotests`, в котором собраны все тесты
- каждая функция проверяет отдельный сценарий
- код запускается из терминала, тесты проходят без участия человека

**Что сделано:**

- проверка цены Samsung galaxy s6
- проверка количества телефонов в категории Phones
- проверка цен Samsung и Nokia с возвратом на предыдущую страницу
- добавление товара в корзину с обработкой всплывающего окна
- переход в корзину и проверка наличия товара
- удаление товара из корзины и проверка, что он исчез
